Exercício 1 — Isto é um algoritmo válido?

a. "Some dois números."
Resposta: Não é um algoritmo válido, pois é ambíguo. Não informa quais números serão somados nem o que deve ser feito com o resultado.

b. "Leia um número. Se for positivo, mostre 'positivo'. Senão, mostre 'negativo'."
Resposta: É um algoritmo válido, pois é finito, ordenado e sem ambiguidade.

c. "Fique repetindo até acabar."
Resposta: Não é um algoritmo válido, pois não é finito e é ambíguo, já que não informa quando a repetição deve terminar.

d. "Pegue um ingrediente qualquer e cozinhe."
Resposta: Não é um algoritmo válido, pois é ambíguo, já que não informa qual ingrediente deve ser utilizado nem como ele deve ser cozinhado.
